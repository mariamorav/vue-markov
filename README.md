# vue-markov
Proyecto web para solucionar ejercicios de probabilidad según el modelo de cadenas de Markov.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
